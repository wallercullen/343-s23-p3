Description

This app allows users to search for a location and will give them Spotify playlist recommendations based on the current weather in that location

Requirements

  • This app is hosted publicly on github.io
  • The site uses media queries to change the layout to grid on wider viewports
  • The app makes use of Spotify's Web API and Weather API to find Spotify playlists based on weather
  • There is a search bar to search for a location to query Weather API
  • Playlists are displayed after searching for a location
